## O Que É um QR Code Estático?

Um QR Code estático codifica os dados directamente no padrão visual. O URL, texto ou credenciais Wi-Fi ficam permanentemente incorporados na imagem. Uma vez criado, o conteúdo não pode ser alterado — para actualizar o destino, é necessário gerar um novo código.

**Características:**
- Grátis
- Sem dependência de servidor
- Sem data de expiração
- Sem rastreamento de digitalizações
- Privacidade máxima

## O Que É um QR Code Dinâmico?

Um QR Code dinâmico codifica um URL de redireccionamento curto. Quando digitalizado, o utilizador é redireccionado através de um servidor externo que pode registar a digitalização e encaminhar para o destino real.

**Características:**
- Requer subscrição paga
- Depende de servidor externo
- Oferece análise de digitalizações
- O destino pode ser alterado após impressão
- Menos privado

## Quando Usar QR Codes Estáticos

- **Cartões de visita:** Os dados de contacto raramente mudam.
- **Embalagens de produtos:** O URL da página do produto é permanente.
- **Sinalética:** A informação é fixa.
- **Partilha de Wi-Fi:** As credenciais são codificadas localmente.

## Quando Usar QR Codes Dinâmicos

- **Campanhas de marketing:** Precisa de testar landing pages.
- **Promoções temporárias:** O destino muda semanalmente.
- **Decisões baseadas em dados:** Precisa de contagem e localização de digitalizações.
